ISupportExporterProgress Interface Reference

Detailed Description

An optional interface supported by an exporter.

If an exporter supports this interface, then the IProgressCB parameter of the ISketchUpExporter::DoExport() method can be non-NULL. If the exporter does not support this interface, that parameter is guaranteed to be NULL.

An exporter supporting this interface will not launch any dialogs or UI elements, as the progress callback mechanism used by SketchUp will interfere and confuse the user.

If this interface is supported, then the exporter can report back on progress.
