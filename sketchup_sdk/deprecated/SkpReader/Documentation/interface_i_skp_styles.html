IUnknown ISkpStyles::_NewEnum [get]

Gets an IEnumVARIANT from the collection.

This will return an object implementing IEnumVARIANT. Each entry in the IEnumVARIANT will be an ISkpStyle object.

Return values:
pVal The IEnumVARIANT interface
Returns:
S_OK : if successful
E_POINTER : if pVal == NULL
E_UNEXPECTED : indicates unexpected internal errors
