LO_RESULT LOGroupCreate ( LOGroupRef *  group,
LOEntityListRef  entity_list 
)

Creates a new group object, populating it with the entities in the given LOEntityListRef. It is possible to create a group from entities that are already in a document, as well as from entities that are not yet in a document. For entities that are already in a document, they can only be grouped if they all belong to shared layers, or all belong to non-shared layers on the same page. If the entities are in a document, then the new group will be added to the document at the top of the group hierarchy.

Parameters
[out]groupThe group object.
[in]entity_listThe list of entities to add to the group.
Returns
  • SU_ERROR_NONE on success
  • SU_ERROR_NULL_POINTER_OUTPUT if group is NULL
  • SU_ERROR_OVERWRITE_VALID if *group already refers to a valid object
  • SU_ERROR_INVALID_INPUT if entity_list does not refer to a valid object
  • SU_ERROR_GENERIC if entity_list is empty
  • SU_ERROR_GENERIC if entity_list contains entities that belong to different documents
  • SU_ERROR_GENERIC if entity_list contains a mix of entities that belong to a document and entities that don't belong to a document
  • SU_ERROR_GENERIC if entity_list contains entities on both shared and non-shared layers, or on non-shared layers belonging to different pages.
  • SU_ERROR_GENERIC if entity_list contains the same entity more than once.

)

Sets the units format on a group.

Since
LayOut 2018, API 3.0
Parameters
[in]groupThe group object.
[in]unitsThe units format to use.
[in]precisionThe units precision. This is expressed as a value in the specified units. LayOut only allows for a finite set of precision values for each units setting, so it will set the precision to the closest valid setting for the specified units. See the "Units" section of LayOut's "Document Setup" dialog for a reference of the available precisions for each units setting.
Returns
  • SU_ERROR_NONE on success
  • SU_ERROR_INVALID_INPUT if group does not refer to a valid object
  • SU_ERROR_LAYER_LOCKED if group is on a locked layer
  • SU_ERROR_ENTITY_LOCKED if group is locked
  • SU_ERROR_UNSUPPORTED if group does not have a scale factor set.
