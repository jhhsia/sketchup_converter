LO_RESULT LOEntityApplyTransform ( LOEntityRef  entity,
const LOTransformMatrix2D *  transform_matrix 
)

Applies a transformation to an entity. This works for all types of entities, regardless of whether or not they have an explicit transformation. It even works on groups, in which case the transformation is applied to everything inside the group.

Returns
  • SU_ERROR_NONE on success
  • SU_ERROR_INVALID_INPUT if entity does not refer to a valid object
  • SU_ERROR_LAYER_LOCKED if entity is on a locked layer
  • SU_ERROR_ENTITY_LOCKED if entity is locked
  • SU_ERROR_NULL_POINTER_INPUT if transform_matrix is NULL

LO_RESULT LOEntityListMoveToLayer ( LOEntityListRef  entities,
LOLayerRef  layer,
LOPageListRef  pages 
)

Moves a list of entities to the given layer. If the layer is non-shared and any entity is on a shared layer, pages must be valid and populated with the pages to move the entities to. In all other cases, pages may be an invalid object. The entities must belong to the same document as the layer and pages.

Parameters
[in]entitiesThe entity object.
[in]layerThe layer definition object.
[in]pagesThe page list object.
Returns
  • SU_ERROR_NONE on success
  • SU_ERROR_INVALID_INPUT if entities does not refer to a valid object
  • SU_ERROR_INVALID_INPUT if entities is empty
  • SU_ERROR_INVALID_INPUT if entities contains the same entity reference more than once
  • SU_ERROR_INVALID_INPUT if layer does not refer to a valid object
  • SU_ERROR_INVALID_INPUT if pages does not refer to a valid object and layer is non-shared
  • SU_ERROR_INVALID_INPUT if pages is empty and layer is non-shared
  • SU_ERROR_INVALID_INPUT if pages contains the same page reference more than once and layer is non-shared
  • SU_ERROR_GENERIC if the entities, layer, and pages are not all in the same document
  • SU_ERROR_LAYER_LOCKED if layer is locked or if any entity is currently on a locked layer
LO_RESULT LOEntityMoveToGroup ( LOEntityRef  entity,
LOGroupRef  group 
)

Moves an entity into a group. If the entity is already in a group, it will be removed from that group prior to being added to the new group. If this action results in the old group containing only one entity, the old group will be collapsed and the remaining entity will be moved to the old group's parent.

Parameters
[in]entityThe entity object.
[in]groupThe group object.
Returns
  • SU_ERROR_NONE on success
  • SU_ERROR_INVALID_INPUT if entity does not refer to a valid object
  • SU_ERROR_LAYER_LOCKED if entity is on a locked layer
  • SU_ERROR_ENTITY_LOCKED if entity is locked
  • SU_ERROR_INVALID_INPUT if group does not refer to a valid object
  • SU_ERROR_GENERIC if entity and group are not in the same document
  • SU_ERROR_GENERIC if entity and group belong to different pages, or if one is shared and the other is not
