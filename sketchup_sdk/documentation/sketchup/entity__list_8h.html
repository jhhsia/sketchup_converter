SU_RESULT SUEntityListBegin ( SUEntityListRef  list,
SUEntityListIteratorRef *  iterator 
)

Sets the iterator reference to the beginning of the list. The given iterator object must have been constructed using SUEntityListIteratorCreate. The iterator must be released using SUEntityListIteratorRelease when it is no longer needed.

Since
SketchUp 2018, API 6.0
Parameters
[in]listThe list.
[out]iteratorAn iterator Ref reference the beginning of the list.
Returns
  • SU_ERROR_NONE on success
  • SU_ERROR_INVALID_INPUT if list is not a valid object
  • SU_ERROR_NULL_POINTER_OUTPUT if iterator is NULL
  • SU_ERROR_INVALID_OUTPUT if *iterator is not a valid object
