Performs the conversion from an active SketchUp model to the target output file using options set during the ShowOptionsDialog method.

Parameters
[in]documentThe ISkpDocument pointer for the model being exported.
[in]outputThe requested output file in UTF-8.
[in]progressThe exporter should support this progress interface. See that for details. The method should be tolerant to a NULL progress callback.
Returns
  • true on success
  • false on failure or cancellation
